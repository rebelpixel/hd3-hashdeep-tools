HD3: Hashdeep Tools
===

A set of cli scripts written for PHP, to be used with `hashdeep` to easily add file integrity protection for a directory of files. These scripts make `hashdeep`'s output a bit easier to read for humans, nothing more.

This is for my personal use and will most likely be not useful to most, but it's here anyway.

Note: This does not provide versioning for backups, use `git` or `restic` for that.
